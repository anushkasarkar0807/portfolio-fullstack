IoT Based Smart Dustbin

A prototype smart dustbin system that opens and closes automatically using motion sensing, measures garbage level via an ultrasonic sensor, detects foul smell with an MQ135 gas sensor, and streams data to the BLYNK IoT platform to trigger notifications when thresholds are exceeded.

Overview

  • Domain: IoT, Embedded Systems
  • Key Sensors: Ultrasonic, MQ135
  • Connectivity: Blynk (mobile & web dashboards)
Publication Link

Features

  • Auto open/close lid via motion/proximity detection.
  • Ultrasonic sensor tracks dustbin fill level.
  • MQ135 monitors foul smell/air quality.
  • Real-time IoT dashboard using Blynk.

How It Works

  • Proximity is detected near the bin, triggering the servo to open the lid.
  • The ultrasonic sensor periodically measures distance to compute fill level.
  • MQ135 reads air quality; spikes imply odor presence.
  • Data is published to Blynk for visualization and alerts.
